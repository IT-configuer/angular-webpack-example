Explore ionic with webpack
---

## Dependencies

- node^4.2.2
- npm^3.4.0
- webpack^1.12.9
- webpack-dev-server^1.14.0

## Getting Started

- npm install -g webpack
- npm install -g webpack-dev-server
- npm install
- npm start
- visit http://localhost:8080/webpack-dev-server/
